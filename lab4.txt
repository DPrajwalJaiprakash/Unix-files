3. Write a shell script to display student's grade( above 90-A, 80-B, 70-C, 60-D)
answer:

echo "enter your grade"
read grade
if test $grade -gt 90
then echo "A"
elif test $grade -gt 80
then echo "B"
elif test $grade -gt 70
then echo "c"
elif test $grade -gt 60
then echo "D"
else echo "Fail"
fi
